# TR
- 'iconImage' klasöründeki 'Contents.json' dosyasını silmeyin!
- İstediğimiz ikonu istediğimiz şekilde kare olarak kesiyoruz ve 'ic_launcher.png' olarak kaydediyoruz.
- Bu kaydettiğimiz 'ic_launcher.png' dosyasını uygulamamızın 'IconImage' klasörüne kopyalıyoruz.
- Uygulamayı çalıştırın.
- Gerekli dosya ve klasörleri 'temp' klasöründe bulacaksınız.

# EN
- Do not delete the 'Contents.json' file in the 'iconImage' folder!
- We cut the desired icon as a square and save it as 'ic_launcher.png'.
- We copy the 'ic_launcher.png' file we saved to the 'IconImage' folder of our application.
- Run the application.
- You will find the necessary files and folders in the 'temp' folder.
